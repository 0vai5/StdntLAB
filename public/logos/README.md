# Meeting Platform Logos

Place your meeting platform logo images in this folder with the following names:

- `google-meet.svg` (or `.png`, `.jpg`) - For Google Meet links
- `zoom.svg` (or `.png`, `.jpg`) - For Zoom links  
- `teams.svg` (or `.png`, `.jpg`) - For Microsoft Teams links

## Recommended Specifications:
- **Format**: SVG (preferred) or PNG with transparent background
- **Size**: 20x20px to 40x40px (will be displayed at 20px height)
- **Aspect Ratio**: Square (1:1)
- **Background**: Transparent

The logos will be automatically displayed next to meeting links in session cards.
